# Dataset

#### - 국토교통부_한국건설기술연구원 상시/수시 교통량 정보조회서비스 <br/>
공공데이터포털 (https://www.data.go.kr/index.do)
```
[1] 수시 도로유형별 교통량 조회 
[2] 상시 차종별 교통량 조회 
[3] 교통량 측정 지점 조회 
```

`홈 > 교통정보 > 구간별통계정보`

`(고속도로 공공데이터 포털 > 유지관리 > 도로)`
